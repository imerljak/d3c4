# Structurizr Component Demo

This page demonstrates the `<Structurizr>` MDX component from `@d3c4/docusaurus-theme-structurizr`. Use this approach when you want to embed an interactive diagram directly inside an MDX page.

## Component Usage

Import the component and pass your DSL as the `dsl` prop:

```mdx
import Structurizr from '@theme/Structurizr';

<Structurizr dsl={`
workspace "My Workspace" {
  ...
}
`} />
```
